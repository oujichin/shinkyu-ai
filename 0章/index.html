Tメソッドとは？

心の背景を読み解き、患者さんを幸せに導く方法

Tメソッドは、治療家が心の背景を理解し、
患者さんの自己受容を促すための体系的なメソッドです。

複雑な心理療法を習得する必要はありません。
必要なのは、患者さんの心の中で何が起きているのか、
その背景を理解することです。

3つの柱

1

心の5階層理論

  • 認知の歪み（考え方のクセ）
  • スキーマ（無意識の自己否定的イメージ）
  • メタ認知（自分を客観視する力）
  • 心理的視野狭窄（思考の余裕がない状態）
  • 自己受容（今の自分を認めること）
2

心の流れの理解

認知 → 感情 → 心の不調 → 自律神経の乱れ → 身体反応
どこで介入すべきかが見えるようになる

3

心理的解放問診

  • 自己否定の奥にある本音に触れるための問診技術
  • 「初めて言うのですが」という言葉を引き出す再現性のある型
