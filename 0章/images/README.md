## 画像生成後の作業

1. 生成した画像を上記のファイル名で保存
2. 推奨サイズにリサイズ（必要に応じて）
3. `/home/kohei68/project/shinkyu-ai/0章/images/` フォルダに配置
4. 最適化（圧縮）して読み込み速度を向上

## SVG図解について

5-layers-diagram.svg と mind-flow-diagram.svg は、図解なので以下の方法で作成できます：

1. **Figma、Canva、Adobe Illustratorなどのデザインツールで作成**
2. **Mermaid.jsやDraw.ioなどの図解ツールで作成**
3. **AIに依頼して作成**

簡易的には、画像として生成してJPGで配置しても問題ありません。
