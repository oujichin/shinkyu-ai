# Tメソッド講座 LP設計書

## 1. サイト概要

### 1.1 目的
治療家（鍼灸師・整体師など）向けに、心因性症状や不定愁訴に対応できる「Tメソッド講座」を販売するランディングページ

### 1.2 ターゲット
- 鍼灸師、整体師、治療家
- 技術はあるのに心因性症状・不定愁訴に悩む患者への対応に限界を感じている人
- 問診で患者の本音を引き出せないと感じている人
- うつや不安の強い患者への関わり方が分からない人
- 患者の心の背景を理解したい人

### 1.3 販売方法
- 銀行振込（即時決済）
- フォーム申し込み

### セクション4: Tメソッドとは
**目的**: メソッドの概要を分かりやすく伝える

#### コンテンツ
**見出し**:
```
Tメソッドとは？
心の背景を読み解き、患者さんを幸せに導く方法
```

**概要**:
```
Tメソッドは、治療家が心の背景を理解し、
患者さんの自己受容を促すための体系的なメソッドです。

複雑な心理療法を習得する必要はありません。
必要なのは、患者さんの心の中で何が起きているのか、
その背景を理解することです。
```

**3つの柱**:

1. **心の5階層理論**
   - 認知の歪み（考え方のクセ）
   - スキーマ（無意識の自己否定的イメージ）
   - メタ認知（自分を客観視する力）
   - 心理的視野狭窄（思考の余裕がない状態）
   - 自己受容（今の自分を認めること）

2. **心の流れの理解**
   - 認知 → 感情 → 心の不調 → 自律神経の乱れ → 身体反応
   - どこで介入すべきかが見えるようになる

3. **心理的解放問診**
   - 自己否定の奥にある本音に触れるための問診技術
   - 「初めて言うのですが」という言葉を引き出す再現性のある型

**ビジュアル**:
- 心の5階層を図解
- 心の流れを矢印で表現

### セクション5: こんな変化が起きます
**目的**: 学ぶことで得られるベネフィットを具体的に示す

#### コンテンツ
**見出し**:
```
Tメソッドを学ぶと、こんな変化が起きます
```

**3つの変化**:

#### 1. 治療の見立てが変わる
- 認知の歪みや心の背景まで見て、いま何が起きているのかを見立てられる
- 自己否定の原因が見立てられ、どうすれば自己受容に向かえるのかまで整理できる
- 原因不明と言われてきた症状に対しても、明確に説明できるようになる
- 見立てを伝えた瞬間に、患者さんの表情の緊張がほどける

#### 2. 心の姿勢（軸）が変わる
- どのような患者さんと向き合っても、心が揺れなくなる
- 「分からない」を恥じず、一緒に見極めていける
- 患者さんがどれほど混乱していても、自分の軸を保ちながら対応できる
- 拙速に評価せず、症状の背景と原因を丁寧に見立てられる

#### 3. 患者さんとの関わり方が変わる
- カウンセリングの技術が上がり、対話の流れを意識して組み立てられる
- どんな順番で聴くか、どこで質問するか、どの言葉を拾うかが明確に
- 「初めて言うのですが」「誰にも言ったことがないのですが」という本音を引き出せる
- 表面的な訴えだけで終わらず、症状の背景まで含めて見立てを深められる

**ビジュアル**:
- Before/After の対比表
- または、3つの変化をアイコンで表現
