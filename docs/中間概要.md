0章 はじめに - 患者の幸せを考える治療家を目指して

## 1. はじめに

こんにちは、鍼灸師の豊田です。私は長年の臨床経験と国立大学院での心理学研究を通じて、「豊田メソッド」を体系化してきました。

あなたは日々の臨床で、こんな経験はありませんか？

- 患者さんの身体の症状を一時的に改善しても、何度も同じ症状で戻ってくる
- 検査では異常がないのに、慢性的な痛みや不調を訴える患者さんが増えている
- 「何か大事なものを見逃している」という感覚がある

実は、患者さんが抱える慢性的な不調の7～8割は、西洋医学的な検査では原因が特定できないものです。そして、その多くに「認知の歪み」という心理的要因が関わっています。

東洋医学には古くから「心身一如」という考え方があります。心と体は切り離せないものであり、お互いに深く影響し合っています。私たち治療家が患者さんの「体」だけを診て「心」を無視してしまうと、本当の意味での回復は難しいのです。

## 2. 私の物語 - 豊田メソッドが生まれた背景

私自身も筋肉の病気を抱えながら生きてきました。そして臨床を続ける中で、ある重要な気づきがありました。それは、「どれだけ筋肉や骨格を調整しても、心の状態が整っていなければ、症状は繰り返し現れる」ということです。

例えば、こんな患者さんがいました。

管理職に昇進して間もなく、動悸や過呼吸に悩まされる30代男性。検査では異常がなく「ストレスが原因かもしれない」と言われるだけでした。彼と長時間かけて対話する中で見えてきたのは、「自分には能力がない」「管理職としてこうあるべき」という強い思い込み。この「認知の歪み」が自律神経を乱し、身体症状につながっていたのです。

私たちは彼の認知パターンを少しずつ修正し、「完璧でなくてもいい」「今の自分でいい」と思えるようサポートしました。その結果、自律神経の症状は徐々に落ち着き、仕事のパフォーマンスも向上したのです。

このような経験から、「心理的アプローチを鍼灸治療に組み込む」という豊田メソッドが生まれました。

## 3. 認知の歪みと身体症状の関係

「認知」とは、物事の受け止め方や考え方のことです。私たちの心は「知識（理性）」「感情」「意識（意志）」の3要素からなります。

感情が暴走すると、理性によるコントロールが弱まり、意志や行動力も落ちていきます。長期間この状態が続くと、認知が歪み、自律神経が乱れ、様々な身体症状として現れるのです。

よくある認知の歪みには以下のようなものがあります：

- べき思考：「〜すべき」「〜でなければならない」と考える
- 白黒思考：物事を「完全か失敗か」の二択でしか考えられない
- 自責思考：何か問題が起きると「自分が悪い」と責める

これらの歪んだ認知パターンは、無意識のうちにストレスを生み出し、自律神経のバランスを乱します。その結果、頭痛、腰痛、不眠、めまいなど、様々な身体症状として現れるのです。

## 4. 豊田メソッドの核心：初回60分心理的解放問診

豊田メソッドの最大の特徴は、初回に60分かけて行う「心理的解放問診」です。

なぜそれほど時間をかけるのでしょうか？

それは、患者さんが本音を語り、自分の認知パターンに気づく「心理的解放」の場を作るためです。治療家として「たっぷり時間をかける」ことには、大きな価値があります：

1. 患者理解の徹底：身体的症状だけでなく、生活背景や心理状態まで深く把握できる
2. 患者さんが本音を語れる場づくり：時間に追われない安心感が、治療効果や納得感の向上につながる
3. 心理学的アプローチを組み込める余裕：患者さんの認知パターンや感情を丁寧に探れる
4. 信頼関係とブランド力の育成：「しっかり理解してくれる」という姿勢が、患者さんの信頼感を高める

また、「非日常的な空間」と「あえて距離を保つ関係性」も重要です。美しく整った環境や静かな施術室など、日常と切り離された空間は「セーフハウス」として機能し、患者さんがストレスを一時的に手放しやすくなります。

## 5. 心理学が必要な理由

なぜ鍼灸師や治療家が心理学を学ぶ必要があるのでしょうか？

慢性化した症状の多くは、検査では原因が見つからないことが多いのです。西洋医学では解決できない場合でも、東洋医学的な「心と体のつながり」を重視したアプローチで改善できることがあります。

自律神経の乱れは、ストレスや感情に大きく影響されます。認知の歪みや感情の偏りが自律神経を乱して不調を引き起こすので、心をケアしないと根本治療にはなりません。

心を適切に扱うには専門知識が不可欠です。患者さんが抱えている認知の歪みを見極め、必要なら修正をサポートできるスキルがあると、東洋医学の強みがさらに生きてきます。

私たち治療家は、患者さんの体だけでなく心も見る必要があります。認知の歪みを見つけ、修正するサポートができれば、根本的な改善へと導くことができるのです。
